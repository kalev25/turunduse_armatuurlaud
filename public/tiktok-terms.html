NOOR Dashboard Terms of Service

Last updated: May 22, 2026

NOOR Dashboard is an internal marketing analytics dashboard used by Ilukliinik NOOR to view social media and marketing performance data. Access is limited to authorized team members.

Use of TikTok Data

The dashboard may connect to TikTok with the permission of an authorized TikTok account administrator. TikTok data is used only to display marketing performance metrics for Ilukliinik NOOR, such as profile information and video performance where available through TikTok APIs.

Account Access

Only authorized users may access the dashboard. Users must not share dashboard access, credentials, API tokens, or exported data with unauthorized third parties.

Data Handling

Data shown in the dashboard is used for internal reporting and marketing analysis. The dashboard does not sell TikTok user data or use it for unrelated purposes.

Changes

These terms may be updated when dashboard features or integration requirements change.

Contact

For questions about the dashboard or these terms, contact Ilukliinik NOOR at info@n8r.ee.
